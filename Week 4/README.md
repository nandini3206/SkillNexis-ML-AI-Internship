# Week 4 – Heart Disease Prediction

## BeeSkilled Machine Learning Internship

### Project Overview

This project develops a Machine Learning model to predict heart disease using Logistic Regression. The workflow includes data preprocessing, model training, evaluation, hyperparameter tuning, and model persistence.

### Topics Covered

- Data Cleaning
- Missing Value Handling
- Label Encoding
- Feature Scaling
- Logistic Regression
- Model Evaluation
- Confusion Matrix
- ROC Curve
- Cross Validation
- GridSearchCV
- Model Saving using Joblib
- Model Loading
- Heart Disease Prediction

### Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib

### Files Included

- Heart.csv
- Week4_Heart_Disease_Prediction.ipynb
- heart_disease_model.pkl

### Outcome

Built an end-to-end Heart Disease Prediction model using Logistic Regression. The model was evaluated using multiple performance metrics, optimized using GridSearchCV, and saved for future predictions.
