# Week 3 – Clustering and Dimensionality Reduction

## BeeSkilled Machine Learning Internship

### Project Overview

This project focuses on Unsupervised Machine Learning using the Iris dataset. The objective is to identify natural groupings in the data using K-Means Clustering and reduce the feature dimensions using Principal Component Analysis (PCA) for visualization.

### Topics Covered

- Data Exploration
- Data Cleaning
- Label Encoding
- K-Means Clustering
- Cluster Visualization
- Principal Component Analysis (PCA)
- Explained Variance Analysis
- Cluster Evaluation

### Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

### Files Included

- Iris.csv
- Week3_Clustering_PCA.ipynb

### Outcome

Successfully implemented K-Means Clustering to group Iris flower species and applied PCA to visualize the clustered data in two dimensions.
